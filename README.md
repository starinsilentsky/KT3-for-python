## Задача 1: Разница списков

**Функция:** `array_diff`

**Описание:**  
Реализуйте функцию, которая вычисляет разницу между двумя списками. Функция должна удалять все вхождения элементов из первого списка (a), присутствующие во втором списке (b). Порядок элементов в первом списке должен сохраняться в результате.

**Пример:**  
```python
array_diff([1, 2], [1]) # [2]
array_diff([1, 2, 2, 2, 3], [2]) # [1, 3]
```

## Задача 2: Сумма пар ##

**Функция:** `sum_pairs`

**Описание:**
Дан список целых чисел и одно значение суммы. Верните первые два значения (!анализируйте слева направо!) в порядке их появления, сумма которых образует эту сумму.

Если существует две или более пар с требуемой суммой, то решением является пара, второй элемент которой имеет наименьший индекс.

**Пример:** 

```python
sum_pairs([11, 3, 7, 5],         10)
#              ^--^      3 + 7 = 10
== [3, 7]

sum_pairs([4, 3, 2, 3, 4],         6)
#          ^-----^         4 + 2 = 6  *
#             ^-----^      3 + 3 = 6
#                ^-----^   2 + 4 = 6
#  правильный ответ помечен * поскольку второй элемент имеет наименьший индекс
== [4, 2]

sum_pairs([0, 0, -2, 3], 2)
#  в этом варианте нет решения
== None

sum_pairs([10, 5, 2, 3, 7, 5],         10)
#              ^-----------^   5 + 5 = 10
#                    ^--^      3 + 7 = 10
#  правильный ответ помечен * поскольку второй элемент имеет наименьший индекс
== [3, 7]
```

## Задача 3: Дубликаты ##

**Функция:** `remove_duplicate_ids`

**Описание:**
Дан словарь, в которой каждый ключ представляет собой строковое число, а каждое соответствующее значение — массив символов.

Создайте функцию, которая возвращает таблицу с одинаковыми ключами, но каждый символ должен встречаться в массивах значений только один раз

**Примечание:**

Если два ключа содержат один и тот же символ, их следует сравнивать численно, и больший ключ сохранит этот символ. Именно поэтому в приведенном выше примере массив под ключом "2" содержит "A" и "B", поскольку 2 > 1.

Если в одном и том же массиве обнаружены повторяющиеся символы, следует сохранить первое их появление.

Пример:

```python
# Пример №1
input = {
  "1": ["C", "F", "G"],
  "2": ["A", "B", "C"],
  "3": ["A", "B", "D"],
}

output = {
  "1": ["F", "G"],
  "2": ["C"],
  "3": ["A", "B", "D"],
}

# Пример №2
input = {
  "1": ["C", "F", "G"],
  "2": ["A", "B", "C"],
  "3": ["A", "B", "D"],
}

output = {
  "1": ["F", "G"],
  "2": ["C"],
  "3": ["A", "B", "D"],
}
# Пример №3
input = {
  "432": ["A", "A", "B", "D"],
  "53": ["L", "G", "B", "C"],
  "236": ["L", "A", "X", "G", "H", "X"],
  "11": ["P", "R", "S", "D"],
}

output = {
  "11": ["P", "R", "S"],
  "53": ["C"],
  "236": ["L", "X", "G", "H"],
  "432": ["A", "B", "D"],
}
```

## Задача 4: Перемещение нулей в конец ##

**Функция:** `lazy`

**Описание:**
Требуется написать функцию-декоратор @lazy(n), где n — частота «нормальных» запусков. Например, если n == 4, то после первого успешного запуска следующие три вызова этой функции ничего не будут делать, а затем 5-й запуск снова будет выполняться нормально. (Первый запуск всегда должен быть успешным, за исключением n == -1, который всегда является ленивым). Однако, если n — отрицательное число, то частота инвертируется (т.е. @lazy(-4) означает, что только каждый 4-й запуск является ленивым, остальные — нормальными). Если n == 1, то функция всегда должна быть нормальной, а если n == -1, то функция всегда должна быть ленивой. n == 0 никогда не будет проверяться. 
Примечание: Когда ленивая функция «ничего не делает», это означает, что она немедленно возвращает None. Ни одна строка «обычной» функции не должна выполняться вообще.

Пример:

```python
@lazy(4)
def half(x):
  return x/2

print(half(10)) # 5
print(half(77)) # None
print(half(63)) # None
print(half(2))  # None
print(half(38)) # 19 

@lazy(-3)
def output_str(s):
  print(s)
  
output_str('Foo') # Foo
output_str('Bar') # Bar
output_str('Pikachu') # Ничего не распечататется
output_str('Gla') # Gla
```
